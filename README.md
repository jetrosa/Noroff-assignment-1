# Noroff assignment 1 - Komputer Store

Create a Dynamic Webpage using HTML, Vanilla JS, CSS (full assignment in the assignment_docs folder).
Bootstrap used for additional (optional) layout design/styling. 

Default (sample) wireframe used for modelling the page as shown in 
the assignment document Appendix B.

1) The Bank – store funds and makie bank loans
2) Work – increase earnings, deposit cash into bank balance and pay back loans
3) Laptops – select and display information about the computers (API data)
